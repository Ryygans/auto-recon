# Auto-Recon Tools

Auto-Recon Tools is an open source, bash-language tool. It's suitable for bug bounty hunters and is available in various formats, including: subfinder, gau, nuclei, katana, and httpx.

This tool is designed solely to assist in the legal and ethical bug finding process. I am not responsible if you use it for criminal purposes!
